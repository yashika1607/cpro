# Gen Z / Gen Alpha To-Do List App 🚀

A modern, aesthetic, and gamified productivity app built with React and plain CSS.

## ✨ Features

### Core Functionality
- ✅ Add, edit, delete tasks with smooth animations
- 📝 Each task includes: title, description, priority, category, due date & time
- 🎯 Priority levels: Low (🍃), Medium (⚡), High (🔥)
- 📂 Categories: Study, Work, Personal, Health, Finance
- 🔍 Search and filter by category/priority
- 📌 Pin important tasks
- ✔️ Mark tasks complete with satisfying animations

### Three Screen Layout

#### 1. Home Screen
- Today's tasks overview
- Progress bar showing daily completion
- Daily Top 3 highlighted tasks
- Quick stats (points, badges)
- Streak counter
- Quick Pomodoro start button

#### 2. All Tasks Screen
- Complete task list with filters
- Search functionality
- Category & priority filters
- Three tabs: Today, Upcoming, Completed
- Sort by priority automatically

#### 3. Profile/Settings Screen
- User profile with avatar
- Badge collection display
- Dark/Light mode toggle
- Theme selection (5 themes)
- Name customization
- Avatar selection (8 options)
- Lazy Mode toggle

### Gamification System 🎮

**Points System:**
- Complete high priority task: 15 points
- Complete medium priority task: 10 points
- Complete low priority task: 5 points
- Complete Pomodoro session: 25 points

**Streak Counter:**
- Tracks consecutive days of app usage
- Auto-updates on login

**Badge System:**
- 🌟 First Step - Complete 1 task
- ⭐ Rising Star - Complete 5 tasks
- 💪 Productivity Pro - Complete 10 tasks
- 🔥 On Fire - 3-day streak
- ⚡ Week Warrior - 7-day streak

### Smart Features 🧠

**Pomodoro Timer:**
- 25-minute focus sessions
- Track time spent on tasks
- Optional task association
- Earn bonus points for completion

**Daily Top 3:**
- Automatically highlights your 3 most important tasks for today
- Based on priority sorting

**Lazy Mode:**
- Shows only one task at a time
- Perfect for maximum focus
- Toggle in settings

## 🛠️ Technical Details

**Tech Stack:**
- React 19 (Functional Components + Hooks)
- Plain CSS (No frameworks!)
- Lucide-style SVG icons (custom components)

**State Management:**
- useState for local state
- useEffect for side effects
- useMemo for optimized calculations

**Data Storage:**
- In-memory state (no localStorage as requested)
- Fresh data on each refresh

## 🎯 How to Use

1. **Getting Started:**
   - Open the app
   - Set your name and avatar in Profile
   - Choose your favorite theme

2. **Adding Tasks:**
   - Click the + button (center of nav bar)
   - Fill in title, description
   - Select priority and category
   - Set due date & time
   - Save!

3. **Completing Tasks:**
   - Click the checkbox on any task
   - Watch your points grow!
   - See progress bar update

4. **Using Pomodoro:**
   - Click "Start Focus Session" on home
   - Or access from anywhere
   - Optionally select a task to track
   - Start timer and focus!

5. **Earning Badges:**
   - Complete tasks regularly
   - Maintain your streak
   - Check your badge collection in Profile

## 🎨 Customization

**Change Theme:**
- Go to Profile → Settings → Appearance → Theme
- Choose from 5 beautiful themes

**Toggle Dark Mode:**
- Profile → Settings → Appearance → Dark Mode

**Update Profile:**
- Profile section
- Click on name to edit
- Select new avatar

## 💡 Tips

1. Use the **Daily Top 3** to focus on what matters most
2. Enable **Lazy Mode** when feeling overwhelmed
3. **Pin** critical tasks so they stand out
4. Use **Pomodoro** for deep work sessions
5. Check your **streak** daily for motivation
6. Filter by **High Priority** when you need urgency
